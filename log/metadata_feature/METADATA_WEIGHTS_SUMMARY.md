# 元数据权重应用总结

本文档总结了论文元数据（Metadata）在 Check-Mentor 各个工作流中如何通过权重系统发挥作用，以实现更智能、更具洞察力的分析。

## 1. 核心权重因子

元数据主要贡献了以下两个核心权重因子，它们被用于后续的排序和筛选决策。

### 1.1. 时效性得分 (`recency_score`)

- **计算方式**: 基于论文的 `publication_date` 动态计算。分数范围为 `[0, 1]`。
- **逻辑**:
  - **近期发表**: 论文的发表日期越接近当前日期，得分越高（趋近于 1）。
  - **时间衰减**: 随着时间的推移，分数会非线性下降。例如，一年内的论文可能得分 > 0.9，而五年前的论文得分可能 < 0.5。
- **作用**:
  - **优先处理新研究**: 确保分析能反映领域的最新动态。
  - **量化“新旧”**: 为“选择最新论文”等模糊需求提供了可量化的依据。

### 1.2. 青年学者加成 (`young_scholar_bonus`)

- **计算方式**: 检查 `young_scholar_index` 字段。
- **逻辑**:
  - 如果 `young_scholar_index` >= 0（即论文有青年学者参与），则该论文获得一个固定的加分项。
  - 如果为 -1，则无加成。
- **作用**:
  - **鼓励发现潜力**: 在某些场景下（如寻找本科生项目），有青年学者参与的论文可能更具创新性或更适合入门，因此给予更高优先级。

## 2. 权重在三大工作流中的应用

不同的工作流对权重的侧重点不同，以实现各自的分析目标。

### 2.1. 工作流一: 教授核心贡献分析 (`workflow_contribution`)

- **目标**: 总结教授最具代表性的核心研究贡献。
- **权重策略**:
  - **主要排序依据**: **时效性得分 (`recency_score`)**。
  - **逻辑**: 默认认为教授最新的工作最能代表其当前的研究方向和水平。因此，在论文数量过多需要进行聚类和采样时，会优先从每个主题簇中选择**时效性最高的论文**作为代表。
  - **青年学者加成**: 在此工作流中权重较低或不使用，因为目标是识别教授本人的核心贡献，而非其合作者。

### 2.2. 工作流二: 领域热点问题分析 (`workflow_field_problems`)

- **目标**: 识别当前研究领域的前沿和热点问题。
- **权重策略**:
  - **主要排序依据**: **综合得分**，其中**时效性得分 (`recency_score`)** 占比最高。
  - **逻辑**: 领域热点问题通常体现在最新的研究中。因此，系统会优先分析近期发表的论文，以捕捉最新的研究趋势。
  - **青年学者加成**: 权重较低。

### 2.3. 工作流三: 本科生科研项目建议 (`workflow_undergrad_projects`)

- **目标**: 提出适合本科生参与、具有启发性的科研项目。
- **权重策略**:
  - **主要排序依据**: **综合得分**，由 **时效性得分** 和 **青年学者加成** 共同决定。
  - **逻辑**:
    1. **时效性**: 项目建议应基于相对较新的技术和方向，避免过时。
    2. **青年学者参与**: 有青年学者（尤其是作为主要贡献者）参与的论文，其研究的切入点、复杂度和实现路径可能更适合本科生借鉴和学习。这类论文通常包含更清晰的问题定义和方法论。
  - **应用**: 在进行代表性采样时，会选择综合得分最高的论文，这样的论文既新颖，又可能具有更适合教学和实践的特点。

## 3. 智能聚类中的权重应用

当论文数量超过阈值（例如10篇）并触发智能聚类时，权重系统是挑选**代表性论文**的核心机制。

- **流程**:
  1. **语义聚类**: 首先，LLM 会根据论文摘要或内容将所有论文分成 3-5 个主题簇。
  2. **计算得分**: 系统为每个簇内的所有论文计算其在当前工作流下的综合得分。
  3. **代表性采样**: 从每个簇中，选择**得分最高的一篇论文**作为该主题的代表。
- **效果**:
  - **保证时效性**: 选出的代表性论文总是各个主题中较新的。
  - **兼顾多重目标**: 在本科生项目建议中，能选出既新颖又具启发性的论文。
  - **智能降维**: 将大规模论文集降维为少数几篇高质量、高时效性的代表作，极大提升了分析效率和质量。

## 4. 总结

| 工作流 | 主要权重因子 | 次要权重因子 | 目标导向 |
| :--- | :--- | :--- | :--- |
| **教授贡献分析** | `recency_score` | - | 聚焦最新、最核心的成果 |
| **领域热点分析** | `recency_score` | - | 捕捉前沿趋势 |
| **本科生项目建议** | `recency_score` | `young_scholar_bonus` | 平衡新颖性与启发性 |

通过这套灵活的权重系统，Check-Mentor 能够根据不同的分析目标，动态地调整其对论文的偏好，从而实现了超越简单文本分析的智能化决策。
